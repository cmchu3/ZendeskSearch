Zendesk Search App

Description:
    An app that, given a set of json files, it allows the users to search the data in said files.
    Written in Java 8.

Running the program:
    Java version should be Java 8.
    Easiest way to run the program is to simply run the executable jar file that is in the main folder
        java -jar ZendeskSearch.jar -filepath <path to the folder with input files>

        example: java -jar ZendeskSearch.jar -filepath /Users/cristina/Desktop/ZendeskSearch/input/
